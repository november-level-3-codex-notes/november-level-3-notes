#### Setting up a Node Project
1. Create a new directory to work within: `mkdir your-project-name`
2. Change into that new directory: `cd your-project-name`
3. In your terminal, enter the command: `npm init`
	- It will ask you setup questions:
		- **project name** - the name of your project
		- **version** - default will be 1.0.0
		- **description** - a way to describe what your project does
		- **entry file** - the main file that runs your application, default is `index.js`
		- **test command** - a way to run external testing libraries 
		- **git repository** - leave blank
		- **keywords** - a way to give your project tags (extra information)
		- **author** - who created this project
		- **license** - we will use MIT or ISC  
		- **type** - we will use `module`
		- **Is this OK?** - type `yes`
#### Difference Between `npm init` and `npm init -y`
- `npm init` will ask for all the setup questions listed above
- `npm init -y` skips all the setup questions
