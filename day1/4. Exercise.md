#### What We Built in Lecture

```js
// index.js
const name = "Your Name";
const age = 25;
const favoriteLanguage = "JavaScript";

console.log(name);
console.log(age);
console.log(favoriteLanguage);
```

1. Created a new folder and run `npm init -y`
2. Created an entry file: `index.js`
3. Added variables for your **name**, **age**, and **favorite language**
4. Printed each one with `console.log()`
5. Ran the file with `node index.js`
