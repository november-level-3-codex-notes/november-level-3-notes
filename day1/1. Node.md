#### What is Node?
- Node is a JavaScript **runtime environment** that allows you to run JavaScript code outside the browser
- We lose access to the DOM and the browser, but gain access to the file system, operating system features

#### What is a Runtime Environment?
- A runtime environment consists of many pieces: 
	1. **JavaScript Engine** - converts JS code into instructions the computer understands
	2. **Built-in Features** - file system access, networking, OS interaction
	3. **Execution Context** - runtime decides where to output your code
		- **Browser** - outputs in devtools
		- **Node** - outputs in terminal 
#### What can we Build with Node?
- Web Servers
- Command Line Tools
- Video Games
