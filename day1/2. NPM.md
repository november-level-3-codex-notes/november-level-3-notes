#### What is NPM?
- NPM (Node Package Manager) is the tool that comes with Node.js and is used to install, manage, and run JavaScript packages
#### What is a JavaScript Package?
- A **package** is reusable code someone else created: libraries and frameworks 
- Comes with Node
	- `node` command runs JavaScript
	- `npm` command manages packages and project tools 
#### What NPM Does
- You can install packages:
```bash
npm install express
```

- You can track installed packages inside:
```
package.json
```

- You can run scripts defined inside `package.json`:
```bash
npm run start
```
#### NPM Init
- `npm init` sets up a new Node project by creating the file that stores your project’s information and settings.
