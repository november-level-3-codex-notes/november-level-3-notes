#### Installing NPM Packages 
- After running the command `npm init` to create a new project, we can then install npm packages
- Visit the NPM Website: https://www.npmjs.com/

#### Figlet 
- https://www.npmjs.com/package/figlet
- Used to create ASCII art
- `npm install figlet`

#### Boxen
- https://www.npmjs.com/package/boxen
- Display text inside a box in console
- `npm install boxen`

#### Chalk
- https://www.npmjs.com/package/chalk
- Used to display colorful text
- `npm install chalk`
