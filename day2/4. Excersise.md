#### Day 2 Exercise 

#### What We Built in Lecture

1. Created a new folder and run `npm init -y`
2. Created an entry file: `index.js`
3. Changed to `"type" : "module"` in `package.json`
4. Installed an npm library ex: `npm install chalk` 
5. Added script: `"start" : "node index.js"`

```js
// sayHello.js
export function hello(){
    return "Hello!";
}
```

```js
// index.js
import chalk from "chalk";
import { hello } from './sayHello.js';

let sayHello = hello();
chalk.blue(sayHello);
```

```json

{
```
