#### What is the Node File System - fs
- It’s a built-in core module that lets you work with files and directories on your computer

#### What you can do with fs
- Read files
- Write files
- Append data to files
- Delete files
- Create/remove folders
- Watch files for changes

#### Note on UTF-8
- Node.js doesn’t assume your file is text - it just reads raw bytes.
- By passing "utf8", you’re telling Node how to interpret those bytes.
- Here’s what changes:
  - With "utf8": you get a string (readable text)
  - Without it: you get a Buffer (raw binary data)

#### File System - Sync vs Async
- fs has two styles:
  1. Synchronous (blocking)
  2. Asynchronous (non-blocking)

#### Synchronous Read, Write, Delete a File
- Reading a file:
- NOTE: file should exist prior to executing the code below
```js
import fs from 'fs';

const data = fs.readFileSync('example.txt', 'utf8');
console.log(data);
```
- Writing (append) to a file:
```js
import fs from 'fs';

fs.appendFileSync('example.txt', 'New content added');

// Behavior:
// file doesn’t exist → created automatically
// file exists → content added at the end
```

- Deleting a file:
```js
import fs from 'fs';

fs.unlinkSync('example.txt');

console.log('File deleted');
```

#### Asynchronous Read, Write, Delete a File
- Reading a file:
```js
import fs from 'fs';

fs.readFile('example.txt', 'utf8', (error, data) => {
  if (error) throw error;
  console.log(data);
});
```
- Writing (append) to a file:
```js
import fs from 'fs';

fs.appendFile('example.txt', 'New content added', (error) => {
  if (error) throw error;
  console.log('Content appended');
});

// Behavior:
// file doesn’t exist → created automatically
// file exists → content added at the end
```

- Delete a file:
```js
import fs from 'fs';

fs.unlink('example.txt', (error) => {
    if (error) {
        console.log('Error deleting file');
        return;
    }
    console.log('File deleted');
});
```

#### Synchronous Folder Operations
```js
import fs from 'fs';

fs.mkdirSync('newFolder'); // creates a folder named "newFolder"
fs.rmdirSync('newFolder'); // removes the folder named "newFolder"
```

#### Asynchronous Folder Operations
```js
import fs from 'fs';

fs.mkdir('newFolder', (error) => {
  if (error) throw error;
  console.log('Folder created');
});

fs.rmdir('newFolder', (error) => {
  if (error) throw error;
  console.log('Folder removed');
});
```
