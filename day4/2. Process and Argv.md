#### What `process` is
- `process` is a global object in Node.js that represents the currently running Node program
- It gives your code access to information about - and control over - the environment your script is running in
- You don’t import it; it already exists

#### What `process.argv` is
- `process.argv` is an array that contains the command-line arguments used when starting a Node program

- When you run: `node index.js hello world`
    - Node stores those values inside `process.argv`

- Structure of `process.argv`
  - `console.log(process.argv);`
  - Output looks like:

  ```js
  [
    '/usr/local/bin/node',
    '/path/to/index.js',
    'hello',
    'world'
  ]
  ```
  | Index | Value                   |
  | ----- | ----------------------- |
  | `0`   | path to Node executable |
  | `1`   | path to script file     |
  | `2+`  | arguments you passed    |

  - So user input starts at index 2
