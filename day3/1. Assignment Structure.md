#### Node Project Structure
- File and folder names should clearly describe what they do
- Good example below:
```js
project/
├── index.js
├── utils/
│   └── math.js
├── package.json
└── .gitignore
```

- Bad example below:
```js
project/
├── stuff.js
├── thing.js
├── finalFINAL.js
```

#### Modules Show Separation of Responsibility
- Bad approach below:
```js
// index.js
function add(a, b) {
  return a + b;
}

console.log(add(2, 3));
```
- Good approach below:
```js
// utils/math.js
export function add(a, b) {
  return a + b;
}
```
```js
// index.js
import { add } from "./utils/math.js";

console.log(add(2, 3));
```

#### `package.json`
- Do not forget to change to `"type" : "module"`
- Do not forget to add your script:
